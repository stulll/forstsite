# Web Design Project with Tailwind CSS

Проект включает верстку первых двух блоков (header и main section) с использованием Tailwind CSS.

## Структура проекта

- `index.html` - главный файл с HTML разметкой и Tailwind CSS стилями

## Компоненты

### 1. Header (Шапка)
- Logo / брендирование
- Навигационное меню (desktop и mobile)
- Кнопка Call-to-Action
- Sticky позиционирование

### 2. Main Block (Основной блок)
- Hero секция с текстом и изображением
- Две кнопки (Primary & Secondary)
- Блок с возможностями (Features section)
- Адаптивный дизайн (Mobile first approach)

## Как использовать

1. Откройте `index.html` в браузере
2. Tailwind CSS загружается из CDN
3. Для продакшена рекомендуется использовать процесс сборки

## Технологии

- HTML5
- Tailwind CSS (CDN)
- Responsive Design (Mobile-First)
